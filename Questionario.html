    1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?
      • R: Para especificar o que deverá ser escrito no campo do formulário
    2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
      • R:Radio: lista de opções com um número limitado de escolhas;
        Checkbox: múltipla escolha ou nenhuma.

      • R: Atribuir como ela funcionará.
    4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
      • R: o atributo name = “”
    5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?
      • R: Para identificar a opção mais apropriada para a tabela.
    6. Na tag <option> para que serve o atributo "selected"?
      • R: Para criar uma caixa de drop-box.
    7. Nas tags de formulário para que serve o atributo "required"?
      • R: Um atributo boolean, para identificar texto dentro do input ou outra tag obrigatória para o formulário.
    8. Para que serve a tag <textarea>?
      • R: Usado para coletar informações extras, sendo uma caixa de texto de múltiplas linhas ou caracteres.
    9. Na tag <form> para que serve o atributo "action"?
      • R: Informa para onde os dados serão enviados.
    10. Na tag <form> para que serve o atributo "method"?
      • R: Informa como será enviado o arquivo, se for como URL ou HTTP.
        • post = “”
